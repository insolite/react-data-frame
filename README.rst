=================
React Frame Table
=================

Table for huge data sets

`Demo <https://insolite.github.io/react-frame-table/>`_

Setup development environment
=============================

#. Install `node` and `gulp`

#. Run in project directory

    .. code-block:: bash

        npm i
        gulp sass
        npm start


#. Open http://127.0.0.1:3030/ in browser
